# Security

If you discover a security vulnerability in Nebula Project software, please report it privately to the security team so we can address it promptly.

Preferred reporting method:

- Email: security@nebula.dev

Please include:

- A description of the vulnerability and affected components
- Steps to reproduce (minimal repro if possible)
- Severity and impact assessment
- Any suggested mitigation or patch

We will acknowledge receipt within 3 business days and work to provide a timeline for a fix. Do not publicly disclose the vulnerability until a coordinated disclosure has been completed.
